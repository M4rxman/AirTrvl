◆ page_size()

size_t mio::page_size ( )
inline

Determines the operating system's page allocation granularity.

On the first call to this function, it invokes the operating system specific syscall to determine the page size, caches the value, and returns it. Any subsequent call to this function serves the cached value, so no further syscalls are made.
